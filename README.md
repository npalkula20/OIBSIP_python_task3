# OIBSIP_python_task2
Basic Weather App using Python

## Objective
The objective of this project is to fetch and display real-time weather information such as temperature, humidity, and weather condition for a given location.

## Steps Performed
1. Took city name as user input.
2. Connected to a weather API.
3. Fetched weather data in JSON format.
4. Parsed and displayed temperature, humidity, and weather condition.
5. Handled invalid user input gracefully.

## Tools Used
- Python
- Weather API
- Visual Studio Code
- GitHub

## Outcome
The application successfully retrieves and displays real-time weather details for a user-specified location.
